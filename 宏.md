# 有用的宏

### 选择最近的掉血超过800的队友施放法术
```
/script 
local i,p;
for i=1,40 do p="raid"..i;
  if CheckInteractDistance(p,4) and UnitHealthMax(p)-UnitHealth(p)>800 then 
    TargetUnit(p);
    CastSpellByName("圣光术(等级 8)");
    TargetLastTarget();
    break;
  end;
end
```

### 牺牲祝福，之后翻页
```
/script local i,p;for i=1,40 do p="raid"..i;if CheckInteractDistance(p,4) then TargetUnit(p);CastSpellByName("牺牲祝福(等级 1)");TargetLastTarget();break;end;end
/script CURRENT_ACTIONBAR_PAGE=2;ChangeActionBarPage();
```
